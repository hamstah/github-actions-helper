# Github Actions Helper

```
usage: github-actions-helper [<flags>] <command> [<args> ...]

Flags:
  --help   Show context-sensitive help (also try --help-long and --help-man).
  --quiet  Do not show any output

Commands:
  help [<command>...]
    Show help.

  issues-comments-create [<flags>]
    Create a comment on a PR

  issues-comments-filter --regex=REGEX [<flags>]
    Filter a comment on a PR

  pulls-get [<flags>]
    Get a PR

  pulls-get-raw --format=FORMAT [<flags>]
    Get a PR in raw format

  pulls-merge --commit-message=COMMIT-MESSAGE [<flags>]
    Merge a PR
```
